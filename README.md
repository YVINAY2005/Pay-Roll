# Pay-Roll Management

A full-stack payroll management application built with React (frontend) and Node.js/Express (backend).

## Deployment on Render

### Frontend Deployment
1. Create a new Static Site on Render
2. Connect your GitHub repository: https://github.com/YVINAY2005/Pay-Roll-Management.git
3. Set the **Root Directory** to `frontend`
4. Build Command: `npm install && npm run build`
5. Publish Directory: `dist`

### Backend Deployment
1. Create a new Web Service on Render
2. Connect your GitHub repository: https://github.com/YVINAY2005/Pay-Roll-Management.git
3. Set the **Root Directory** to `backend`
4. Build Command: `npm install`
5. Start Command: `npm start`

## Local Development

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm start
```

## Features
- User authentication
- Employee management
- Salary slip generation
- Expense tracking
- Dashboard with statistics
